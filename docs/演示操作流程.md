## 演示流程（约 15 分钟）

### 阶段一：系统概览（1 分钟）

**目标**：展示系统整体架构和核心功能模块

**操作步骤**：

1. 打开浏览器，访问 `http://localhost:5173`
2. 使用 admin 账号登录
3. 进入"系统概览"页面
4. 展示：
   - 知识库统计（总数、文档数）
   - 用户统计（总用户数、在线用户数）
   - 最近活动（文档上传、检索记录）
   - 品牌配置效果（系统名称、Logo）

---

### 阶段二：知识库与文档管理（3 分钟）

**目标**：展示文档上传、处理流程和状态管控

**操作步骤**：

1. 进入"知识库管理"页面
2. 展示知识库列表，点击查看详情
3. 进入"文档管理"页面
4. **演示文档上传流程**：
   - 点击"上传文档"按钮
   - 选择 `demo-materials/测试文档1-公司考勤制度.md`
   - 点击上传
   - 观察文档状态从 `pending` → `processing` → `completed`
5. **演示文档状态准入**：
   - 上传一个新文档，在处理中（`processing`）时尝试在命中测试中选择
   - 验证前端显示禁用状态和提示文字

---

### 阶段三：命中率测试（4 分钟）

**目标**：展示三种检索方式的效果对比

**操作步骤**：

1. 进入"命中率测试"页面
2. 选择已处理完成的文档（如"公司考勤制度"）
3. **演示向量检索**：
   - 选择检索模式：向量检索
   - 输入问题："公司年假有几天？"
   - 点击检索
   - 展示检索结果和命中分数
4. **演示全文检索**：
   - 选择检索模式：全文检索
   - 输入同一问题
   - 点击检索
   - 对比与向量检索的差异
5. **演示混合检索**：
   - 选择检索模式：混合检索
   - 输入同一问题
   - 点击检索
   - 展示融合后的结果
6. 测试其他标准问题（Q03、Q05、Q08）

---

### 阶段四：智能对话（3 分钟）

**目标**：展示 RAG 增强的智能对话功能

**操作步骤**：

1. 进入"智能对话"页面
2. 选择知识库（如"公司制度知识库"）
3. **演示基础对话**：
   - 输入问题："怎么申请年假？"
   - 点击发送
   - 展示流式响应效果
   - 展示回答中引用的文档来源
4. **演示多轮对话**：
   - 继续提问："年假可以跨年使用吗？"
   - 展示系统记住上下文
5. **演示上下文引用**：
   - 输入问题："迟到会有什么处罚？"
   - 展示回答正确引用考勤制度内容

---

### 阶段五：用户组权限隔离（2 分钟）

**目标**：展示用户组数据可见性隔离

**操作步骤**：

1. **使用普通用户登录**：
   - 退出 admin 登录
   - 使用 user1（研发组）登录
2. 进入"知识库管理"页面
3. 展示 user1 只能看到"研发知识库"
4. **验证跨组访问阻断**：
   - 尝试直接访问产品组知识库（通过 URL 或 API）
   - 展示 403 权限不足错误
5. **切换用户验证**：
   - 使用 user2（产品组）登录
   - 展示 user2 只能看到"产品知识库"
6. **管理员模式**：
   - 使用 admin 登录
   - 展示 admin 可查看所有知识库

---

### 阶段六：白标配置（OEM）（2 分钟）

**目标**：展示品牌定制功能

**操作步骤**：

1. 使用 admin 登录
2. 进入"品牌配置"页面
3. **修改品牌信息**：
   - 修改系统名称："企业智能助手" → "XX公司专属助手"
   - 修改主题色：选择自定义颜色
   - 点击保存
4. **展示动态渲染**：
   - 刷新页面
   - 展示品牌配置实时生效
5. **验证默认配置兜底**：
   - 清空品牌配置
   - 展示系统使用默认值

## 演示技巧

### 时间控制

| 环节 | 建议时长 | 超时处理 |
|------|:--:|------|
| 系统概览 | 1 分钟 | 跳过细节，直接进入核心功能 |
| 文档管理 | 3 分钟 | 跳过上传过程，直接展示已处理文档 |
| 命中率测试 | 4 分钟 | 重点演示混合检索，其他快速带过 |
| 智能对话 | 3 分钟 | 演示 2-3 个关键问题 |
| 用户组权限 | 2 分钟 | 重点演示跨组阻断效果 |
| 白标配置 | 2 分钟 | 快速修改一项配置展示效果 |

### 常见问题应对

| 问题 | 应对策略 |
|------|------|
| API Key 失效 | 提前准备备用 Key，演示前验证 |
| Chroma 连接失败 | 使用降级模式（纯 BM25）继续演示 |
| 文档处理超时 | 提前上传并等待处理完成 |
| 检索结果不理想 | 切换检索模式，展示混合检索效果 |
| 网络延迟 | 说明演示环境限制，展示核心功能 |

## 本地启动

```bash
# 1. 确认 .env（根目录或 backend/）
# 2. 装依赖
cd backend
pip install -r requirements.txt

# 3. 初始化库（在仓库根目录）
cd ..
python scripts/init_db.py

# 4. 另开终端：Chroma（Windows 必须 --host 127.0.0.1）
chroma run --path ./chroma_data --host 127.0.0.1 --port 8000

# 5. 自测
python scripts/self_test_rag.py

# 6. 起后端
cd backend
uvicorn app.main:app --reload --port 8001
```

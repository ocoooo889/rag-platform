## 一、环境要求

| 组件 | 版本要求 | 说明 |
|------|:--:|------|
| Python | 3.10+ | 后端运行环境 |
| Node.js | 18+ | 前端运行环境 |
| Chroma | 0.4+ | 向量数据库 |
| SQLite | 3.30+ | 业务数据库（内置） |

### 1.1 系统依赖

**Linux/Mac：**

```bash
# Ubuntu/Debian
sudo apt-get update && sudo apt-get install -y python3 python3-venv python3-dev nodejs npm

# macOS (使用 Homebrew)
brew install python node
```

**Windows：**

- Python 下载：https://www.python.org/downloads/
- Node.js 下载：https://nodejs.org/

## 二、安装步骤

### 2.1 克隆仓库

```bash
git clone <仓库地址>
cd rag-platform
```

### 2.2 配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，配置以下内容：
# - ENV: 个人环境标识（如 dev-zhangsan）
# - OPENAI_API_KEY: 你的 API Key
# - OPENAI_BASE_URL: API 地址（可选，默认 OpenAI）
```

### 2.3 后端安装

```bash
cd backend

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Linux/Mac
source venv/bin/activate
# Windows
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 2.4 前端安装

```bash
cd frontend

# 安装依赖
npm install
```

## 三、启动流程

### 3.1 一键启动（推荐）

**Linux/Mac：**

```bash
bash scripts/start_all.sh
```

**Windows：**

```bash
scripts\start_all.bat
```

### 3.2 手动启动

#### 步骤 1：启动 Chroma

```bash
chroma run --path ./chroma_data --port 8000
```

#### 步骤 2：启动后端

```bash
cd backend
source venv/bin/activate
uvicorn app.main:app --host 0.0.0.0 --port 8001 --reload
```

#### 步骤 3：启动前端

```bash
cd frontend
npm run dev
```

### 3.3 服务地址

| 服务 | 地址 | 说明 |
|------|------|------|
| 前端页面 | http://localhost:5173 | 管理后台入口 |
| 后端 API | http://localhost:8001 | API 接口 |
| 接口文档 | http://localhost:8001/docs | Swagger 文档 |
| Chroma | http://localhost:8000 | 向量库服务 |

## 五、环境隔离配置

### 5.1 每人独立配置

确保每人 `.env` 文件中以下配置唯一：

```bash
# 个人开发环境标识
ENV=dev-你的姓名拼音

# 个人独立资源路径
LOCAL_DB_NAME=dev_你的姓名拼音_rag.db
CHROMA_COLLECTION_SUFFIX=_你的姓名拼音
UPLOAD_DIR=./uploads/dev_你的姓名拼音
```

### 5.2 隔离效果

| 资源 | 隔离方式 | 效果 |
|------|------|------|
| SQLite 数据库 | 独立文件 | 每人独立数据库 |
| Chroma 集合 | 集合名后缀 | 每人独立向量空间 |
| 文件上传 | 独立目录 | 每人独立上传目录 |

## 六、常见问题

### 6.1 端口占用

```bash
# Linux/Mac 查看端口占用
lsof -i :8000
kill -9 <PID>

# Windows 查看端口占用
netstat -ano | findstr :8000
taskkill /F /PID <PID>
```

### 6.2 依赖安装失败

```bash
# 更新 pip
pip install --upgrade pip

# 清理缓存后重新安装
pip cache purge
pip install -r requirements.txt
```

### 6.3 Chroma 启动失败

```bash
# 确保 chromadb 已安装
pip install chromadb

# 尝试重新启动
chroma run --path ./chroma_data --port 8000
```

### 6.4 前端代理配置

确保 `frontend/vite.config.js` 中代理配置正确：

```javascript
server: {
  proxy: {
    '/api': {
      target: 'http://127.0.0.1:8001',
      changeOrigin: true
    }
  }
}
```

## 三、交付清单

### 3.1 可运行性（P0）

| 交付物 | 说明 |
|--------|------|
| README「最短正确路径」 | clone → checkout → `.env` → venv+pip → Chroma → uvicorn → 前端 |
| `start_all.bat/.sh` | 缺 `.env` 提示复制；先依赖后服务；Chroma 带 `--host 127.0.0.1`；心跳轮询 |

## 五、给队友（复制即用）

```bat
git checkout zhangyu-mlbg
copy .env.example .env
:: 填好 OPENAI_API_KEY / ENV / LOCAL_DB_NAME / CHROMA_COLLECTION_SUFFIX / UPLOAD_DIR
:: 确认 CHROMA_HOST=127.0.0.1

:: 首次
scripts\start_all.bat

:: 以后日常
scripts\restart_dev.bat
```

| 地址 | 用途 |
|------|------|
| http://127.0.0.1:5173 | 前端（admin / admin123） |
| http://127.0.0.1:8001/docs | API |
| http://127.0.0.1:8000/api/v2/heartbeat | Chroma 自检 |

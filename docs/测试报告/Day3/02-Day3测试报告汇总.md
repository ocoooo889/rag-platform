## 四、P0 / P1 一览（须优先关闭）

### P0

1. 角色名冲突：`admin`/`user` vs `管理员`/`普通用户`；登录无 user，前端写死管理员  
2. 管理写接口缺少 admin 强制校验（可 API 绕过）  
3. 缺 `GET /api/auth/me`  

### P1

4. 白标静态资源未挂载；Login/Layout 硬编码  
5. 401/403/4003 统一体与语义不一致  
6. Chat 输入 `question`；会话 API 默认关闭  
7. 文档无 3s 轮询；不支持 pdf；路由非嵌套  
8. UserManage 无用户组分配；部分 API 静默 Mock  

完整台账见 `01` 报告 §9（BUG-D3-01～BUG-D3-12）。

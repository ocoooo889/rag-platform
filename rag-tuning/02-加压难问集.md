# 加压难问集（基于 3 篇 demo 文档）

> 文档：考勤制度 / 差旅报销 / RAG 架构设计  
> ★ = 第 0 步至少必测

---

## H01 ★ 同义改写（考勤）

- **问句：** 带薪休假天数怎么算？工作满一年能休几天？  
- **预期关键词：** 年假、5 天、满 1 年  
- **测什么：** 不问「年假」三字，keyword 是否变差、hybrid/vector 能否救回来  

## H02 ★ 同义改写（差旅）

- **问句：** 普通员工飞北京住酒店一晚最多能报多少？  
- **预期关键词：** 一线 / 500 / 住宿  
- **测什么：** 「酒店」对「住宿」同义；数值是否进 Top3  

## H03 ★ 跨段综合（考勤）

- **问句：** 事假扣钱吗？和年假比有什么不一样？  
- **预期关键词：** 无薪、带薪、扣工资  
- **测什么：** 答案跨多段时 top_n=3 是否够用  

## H04 ★ 指代多轮（重点：Query Rewrite）

1. 先问：`公司年假有几天？`（完整问，应能答）  
2. 再问（同 session）：`那它怎么申请？`  
- **预期：** 第二问应检索到「提前 3 个工作日 / 上级审批」  
- **改前常见失败：** 只搜「它」「申请」，命中无关或空 → 对话瞎编或未查到  
- **测什么：** 历史未进检索的真实缺口  

## H05 ★ 文档没有（防幻觉）

- **问句：** 公司有没有陪产假？天数多少？  
- **预期：** 参考片段无陪产假 → 对话应倾向「未查询到」类，而不是编造天数  
- **测什么：** 调优后幻觉是否变多（变多=失败）  

## H06 数值细节（差旅）

- **问句：** 总监去北京住一晚限额是多少？  
- **预期关键词：** 1000  
- **测什么：** 职级+城市精确匹配  

## H07 干扰串库

- 同一 KB 放入考勤+差旅两篇后问：`北京出差住宿标准`  
- **预期：** 命中差旅，不要被考勤「工作时间」等噪音顶掉  

## H08 指代多轮（架构文档）

1. 先问：`混合检索的融合公式是什么？`  
2. 再问：`那 Embedding 挂了怎么办？`  
- **预期第二问：** 降级 BM25 / 关键词检索
